# Getting Started

## IOS Development

Enter in directory of ios:
```
cd ios
```

and run `flutter run`

## Android Development

Enter in directory of android:
```
cd android
```

and run `flutter run`

# To create a new package:

Enter in directory `/packages` and run:
```
flutter create template=package package_name
```
